Earth Shattering Balls

This is a typing game where more and more balls hurtle towards Earth at continously increasing speeds. The user is tasked with trying to save Earth by typing the first letter of the color of the ball (Red = 'R', Blue = 'B', etc.) to remove and prevent it from hitting Earth. Each incorrect letter typed results in another ball flying towards Earth. Every ball that hits Earth deducts points and if your score drops below zero the game ends.

Features: Java, GUI, Event Handlers, Exception Handling

Reversi

This is a game where the users objective is to have the most squares occupied by white circles at the end of the game. A move is only valid if the unoccupied square is horizontal, vertical, or diagnal of another square occupied by your circle, all squares between the desired move and an existing occupied square of yours must be occupied by your opponent (the computer). When a valid move is made, all of your opponents squares between your existing square and the new square are flipped and now belong to you. The game ends when all squares have been filled or no other valid moves exist.

Features: Java, GUI, AI, Event Handlers, Exception Handling
